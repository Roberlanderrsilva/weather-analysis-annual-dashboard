# 🌦️ Weather Data Pipeline & Analytics Framework

![Dashboard Anual](https://raw.githubusercontent.com/Roberlanderrsilva/weather-analysis-annual-dashboard/main/dashboard_anual.png)


## 📌 Visão Geral
Este framework representa a convergência entre **Engenharia de Dados** e **Gestão de Riscos**. O objetivo principal é o desenvolvimento de um pipeline automatizado que transforma dados climáticos brutos em inteligência analítica para suporte à decisão em setores como logística e agronegócio.

### O que este projeto resolve:
* **Ingestão Analítica:** Processamento de séries temporais históricas (365 dias) simulando variações estocásticas reais.
* **Transformação de Dados:** Implementação de médias móveis para filtragem de ruídos térmicos e identificação de tendências sazonais.
* **Inteligência Visual:** Geração automática de dashboards multivariados (Linhas, Barras e Distribuição Setorial) para diferentes níveis de visualização (Operacional e Gerencial).


## ⚖️ Conformidade e LGPD (Data Privacy)
Estruturado seguindo as diretrizes da **LGPD**:
* **Anonimização:** Dados sintéticos sem PII.
* **Finalidade:** Estudos estatísticos climáticos.
* **Transparência:** Processamento local (On-premise).

## 🛠️ Arquitetura e Engenharia
* **Ambiente:** Lubuntu Linux (SSD/Swap optimized)
* **Linguagem:** Python 3.12+
