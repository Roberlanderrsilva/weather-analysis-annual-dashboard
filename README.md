# 🌦️ Weather Data Pipeline & Analytics Framework

![Dashboard Anual](https://raw.githubusercontent.com/Roberlanderrsilva/weather-analysis-annual-dashboard/main/dashboard_anual.png)

## 📌 Visão Geral
Este projeto simula um pipeline de dados climáticos para análise de sazonalidade e risco ambiental. Desenvolvido sob princípios de **Engenharia de Software**, focando em escalabilidade e governança de dados.

## ⚖️ Conformidade e LGPD (Data Privacy)
Estruturado seguindo as diretrizes da **LGPD**:
* **Anonimização:** Dados sintéticos sem PII.
* **Finalidade:** Estudos estatísticos climáticos.
* **Transparência:** Processamento local (On-premise).

## 🛠️ Arquitetura e Engenharia
* **Ambiente:** Lubuntu Linux (SSD/Swap optimized)
* **Linguagem:** Python 3.12+
